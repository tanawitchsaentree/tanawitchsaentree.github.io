## Overview

A commercial claims platform handling high-value property and liability cases. The job changed shape — instead of shipping Figma, I started shipping verified, working software straight from the ticket.

## What I Did

- Built the closure-automation flow, skeleton conversion redesign, and policy selector component from acceptance criteria
- Established an audit-build-verify loop separating the build prompt from the verification pass
- Extracted failure patterns into a skill library to prevent repeated debugging

## The Outcome

Features landed the same day tickets arrived. The prototype became the backend API contract. The design artifact was the running software.
