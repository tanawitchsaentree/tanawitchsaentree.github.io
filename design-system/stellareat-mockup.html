Mockup notes — direction check only

  • Illustrations: SVG watercolor approximation — flat washes via layered ellipses. In final build, replace with actual commissioned or AI-generated Niemann-style illustrations.
  • Glass panel: backdrop-filter blur(24px) + rgba(255,255,255,0.72). Requires the scattered illustration background to read correctly — tested here.
  • Breadcrumb nav: "Fridge → Equipment → Preferences → Recipes" — underline on active, not circles.
  • Selection state: Each item category maps to an accent color. Protein = tomato red, produce = sage green, dairy = sky blue, pantry/fat = mustard. Consistent across grid.
  • No generate button: "Equipment →" advances the flow. The AI hasn't been invoked yet — this is information gathering.
  • Font: PP Editorial New Italic for the panel title. Feels like a handwritten cookbook question, not a UI label.
  • Not built yet: Steps 2–4 (equipment, preferences, AI reasoning + recipes). Awaiting direction approval on Step 1 before continuing.
